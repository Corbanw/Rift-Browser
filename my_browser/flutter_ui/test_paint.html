Paint Renderer Test Page

This is a test page for the new paint renderer that directly paints HTML and CSS to a canvas instead of converting to Flutter widgets.

Features being tested:

  • Direct HTML/CSS painting
  • Text rendering
  • Background colors
  • Layout and positioning
  • Scroll support

The paint renderer should display this content by converting the HTML and CSS into draw commands and painting them directly to a Flutter canvas.

This approach should be more efficient for complex web pages as it bypasses the Flutter widget tree and renders directly to the graphics layer.
